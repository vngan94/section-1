# Getting Started


## Installation

npm install


## Run Server


```bash
npm start
```

Server runs on: `http://localhost:3005`

```

## Stop Server


Press `Ctrl + C`


## Available Commands


| Command | Description |
|---------|-------------|
| `npm start` | Start server |
| `npm test` | Run tests |


## Endpoints

- `GET /api/status`
- `POST /api/feedback`
- `GET /api/feedback`
